Overview

Homely is a creative, mobile-friendly real estate template suitable for a wide range of sites. Anything from large, robust property listing sites, to local real estate agencies, Homely is a perfect fit. Use our pre-made templates to quickly get up and running, or easily mix and match modules to create unique pages. Homely is built on the Twitter Bootstrap framework and includes 35 HTML page templates.

Main Features:

  • 7 Home Page Variations
  • Grid, list and map property layouts
  • Property submission and user profile templates
  • Google Maps Integration
  • Grid and list agent layouts
  • Fully Responsive (powered by Bootstrap 3)
  • Working AJAX contact form
  • Crossbrowser Compatible
  • Clean, organized, well-commented code

HTML Structure

Homely's HTML5 pages are broken down into logical sections using the <section> tag. Each section has it's own container that holds the content for that section. A few sections do not use the <section> tag because they have more semantic HTML5 tags such as the <header> or <footer> tag.

For example, take a look at the header section:

<header>
<div class="container">
// content goes here
</div>
</header>

Anything that is put inside the container class will have a fixed width and be centered on the page. This is where you will want to put your content. Anything placed outside of the container will not be fixed and centered. For instance, if you wanted to put a background image that stretches the full width of the header you would apply a css style to the <header> tag (not the container).

Header background image example:

header {
background: url('../images/headerBg.png');
background-size: cover;
}

Within each section container, you will notice “col-lg-x” class occurring quite frequently. These classes are used to build the fundamental building blocks of Homely using the bootstrap framework. Col-lg-6 means that section will be 6 columns wide (out of 12).

For example, say you want a section to have three vertical columns of content. Each content block would have to be 4 columns wide (12 divided by 3 equals 4). Take a look at the following code:

<section id= "yourSection">
<div class = "container">
<div class= "row">
<div class= "col-lg-4">
Your content here
</div>
<div class= "col-lg-4">
Your content here
</div>
<div class= "col-lg-4">
Your content here
</div>
</div>
</div>
</section>


CSS Structure

Homely uses 9 external stylesheets total, however, you only have to worry about the main stylesheet (style.css). Unless you want to customize the look of a Jquery plugin.

At the top of the main stylesheet(style.css), there is a very simple reset, followed by the global styles. Global styles are any styles that are not specific to a certain section. This is where you define the things like the font family, color and size that will be applied to the entire site. Each section in the html files has a corresponding section in the main stylesheet where all of the elements within that section are styled. For instance, the style starts with the header section, then the sub-header sections (home, map, and page) and so on. The CSS sections are ordered in the same order as they are found in the html file.

If you want to customize the responsive qualities of Homely, you will want to edit the responsive.css file. Note that a majority of the themes responsive qualities are being inherited from the bootstrap framework. To customize these qualities you will have to refer to the bootstrap documentation.

JavaScript Structure

All Javascript imports and script can be found at the bottom of the HTML pages.

Homely utilizes 15 external JavaScript files:

  1. JQuery
  2. JQuery UI
  3. Bootstrap
  4. Isotope
  5. HTML5 Shiv
  6. Map
  7. Map Search
  8. Map Single
  9. Nouislider
  10. wNumb
  11. Respond
  12. Chosen
  13. html5 lightbox
  14. Slick Slider
  15. Global

1. JQuery (jquery.js)
JQuery is a JavaScript library that reduces the amount of code required to make the site function. JQuery is required for the site to function properly.
Source: http://jquery.com/

2. JQuery UI (jquery-ui.min.js)
JQuery UI is a JavaScript library for jQuery that assists with interactive interface elements, such as tabs and accodions.
Source: https://jqueryui.com/

3. Bootstrap (bootstrap.min.js)
Bootstrap is a front-end framework that enables faster and more efficient web development. Bootstrap is required for the site to function properly.
Source: http://getbootstrap.com/

4. Isotope (isotope.min.js)
Isotope is a jquery plugin used to layout and filter items. In Easy Living, it is used on the property listing masonry page.
Source: http://isotope.metafizzy.co/

5. HTML5 Shiv (html5shiv.min.js)
The HTML5 Shiv is unique script because it is called at the top of the page. It is conditionally loaded if Internet Explorer 9 or below is being used. It's main purpose is to preserve HTML5 code in older browsers.
Source: https://github.com/aFarkas/html5shiv

6. Map (map.js)
map.js is a custom written script that interacts with the Google Maps API. This script is used to display any section of the template that involves a map with multiple markers.

7. Map Search (map-search.js)
This script is almost identical to map.js, however it is used to display a map with only one marker and a search input. For example, it is used on the property submit page.

8. Map Single (map-single.js)
This script is almost identical to map.js, however it is used to display a map with only one marker. For example, it is used on property single pages.

9. Nouislider (nouislider.min.js)
The nouislider script is used specifically for the price range slider within the filters on the home page and home page map.
Source: http://refreshless.com/nouislider/

10. wNumb (wNumb.js)
wNumb is a script that allows the numbers in the price sliders to be properly formatted.
Source: https://refreshless.com/wnumb/

11. Respond (respond.min.js)
Similar to HTML5 Shiv, Respond is called conditionally at the top of the page for IE 9 or below. The purpose of this script is to allow CSS3 media queries to function in older browsers.
Source: https://github.com/scottjehl/Respond

12. Chosen (chosen.jquery.min.js)
Chosen is a jQuery plugin that makes long, unwieldy select boxes much more user-friendly.
Source: https://harvesthq.github.io/chosen/

13. html5 lightbox (html5lightbox.js)
jQuery Image and Video LightBox Plugin- supports images, Flash SWF files, YouTube, Vimeo and local mp4, m4v, flv, ogg, ogv and webm videos, works on Windows, Linux, Mac, iPhone, iPad, Android, Windows Phone and all modern web browsers
Source: https://html5box.com/html5lightbox/

14. Slick Slider (slick.min.js)
Slick slider is a fully responsive, jquery slider plugin. It is used for all sliders across the theme.
Source: http://kenwheeler.github.io/slick/

15. Global (global.js)
Global is the main custom script used in Homely. This is where all plugins get ininiated and where all plugin options are set.
